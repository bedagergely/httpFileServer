## build
`javac *.java`

## run class files
`java App`

## create runnable jar
`jar cfe httpFileServer.jar App *.class`

## run jar
`java -jar httpFileServer.jar`

## clean folder
`rm *.class *.jar`

## Use
Open browser and search your html file at localhost:8080/[HTML FILE NAME]

## Stop
Open browser and go to localhost:8080/stop
